# RacePal — App Use Cases

## Authentication
- Log in / log out

## Profile
- Add a profile picture
- Add a short bio that includes:
  - Favourite race
  - Favourite tune
  - Favourite distance to run
- Show flags of the countries where the runner has completed runs

## Race Discovery (main use case)
- Search for a race by: name, location, distance, date range
- See what people you know (pals) thought of them
- Search results show: race name, upcoming date, average rating with count, names of pals who have done it
- Results can be viewed as a list or on a map

## Race Management
- Add a race manually
- Mark a race as "planning to attend"
- Mark a race as something done in the past
- Encouraged to add a review after attending

## Reviews
- Rating scale: 1 to 5
- Captures: title, description, rating, photos, whether they would recommend it, date the run was completed
- Only mandatory field: rating

## Social / Pals
- Find friends via search
- Follow other runners; control who can follow you
- When follows go both ways → become "Pals"
- View a calendar with own races (upcoming and past)
- Toggle to show Pals' races on the calendar as well

## Scope
- "Races" includes races and parkruns
- Must support all major UK races
- Must support all UK parkruns
- Must support parkruns abroad (runners travel internationally)
