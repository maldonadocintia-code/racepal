# RacePals — Project Description

An app that allows runners to decide on their next event based on experiences from their running pals and network, by publishing reviews.

Runners can plan runs and share a calendar. The runs can be parkruns or races. Runners connect as **Pals** (a mutual friendship — request and accept).

**Name:** RacePals (display name; package/repo/Firebase keep the original `racepal`)
**Scope:** Lightweight Android app, sideloaded APK (beta)
